Ruby Tanbo 1.0 Installation Instructions

Windows XP

  1. Install Ruby 1.8.6 using the Windows one click installer, following these steps:
    1. Go to the file download page for the Windows Ruby one click installer: http://rubyforge.org/frs/?group_id=167
    2. Download ‘ruby186-26.exe’: http://rubyforge.org/frs/download.php/29263/ruby186-26.exe
    3. Run the installer. You can deselect ‘SCitE’ (it’s a text editor to help you program ruby).
  2. Install `msvcp711.dll` and `msvcr711.dll`. They are C and C++ runtimes for windows (the wxRuby distribution needs msvcp71.dll, see the discussion at http://www.ruby-forum.com/topic/140300#new). Where do you get them, you ask? Sketchy 3rd party sites (Google it. I used dll-files.com). Once you get the files, put them in your `C:\WINDOWS\system32\` folder. If you want, you can get them from me at: msvcp71.zip and msvcr71.zip. I’m going to talk to the wxruby people about how sketchy/bad this is.
  3. Install wxRuby, a Ruby binding to the wxWidgets windowing toolkit, following these steps:
    1. Open a Windows command shell window. The easiest way to do this is “Start Menu → Run…”. Simply type ‘cmd’ and hit okay.
    2. Type ‘gem install wxruby’ and hit enter.
    3. If you are prompted for a version, select the one marked ‘x86-mswin32’. For me, this was option 4.
    4. Check out the sample interaction
    5. Download and extract the Ruby Tanbo .zip file: http://www.boxofmonocles.com/ruby_tanbo_10.zip
    6. You should now be able to double-click app.rb and get a Tanbo board!

Mac OS X 10.5

  1. Start with step 3 above, using /Applications/Utilities/Terminal.app to “sudo gem install wxruby”.
  2. Instead of double-clicking the Tanbo file, navigate to the directory where you extracted the files (using cd Downloads or cd Desktop or something similar) and use the command ruby app.rb .
